Kid Group Class

This class teaches the fundamentals of table tennis to provide kids a solid start in table tennis.

Class contents include but are not limited to:

  • Forehand and backhand strokes
  • Footwork
  • Forehand and backhand rally practices
  • Serving and Receiving
  • Physical conditioning and group games
