Beginner Kid Group Class

This class teachs the fundamentals of table tennis to provide kids a solid start in table tennis.

Class contents include but are not limited to:

  • Forehand and backhand strokes
  • Footwork
  • Forehand and backhand rally practices
  • Serving and Receiving
  • Physical training and group games

Coaching

  • Classes are tought by Swan Head Coach Avishy Schmidt
  • Assistant coaches will be provided with a guaranteed student/coach ratio of 1:4
