Class Content

This class aims to teach the fundamentals of table tennis to provide kids a solid start in table tennis.

Class contents include but are not limited to:

  • Forehand and backhand strokes
  • Footwork
  • Forehand and backhand rally practices
  • Serving and Receiving
  • Physical training and group games
